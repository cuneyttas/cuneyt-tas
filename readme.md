## Kurulum

1. Önce .env.example dosyasından .env dosyası oluştur.
1. Alttaki komutları sırasıyla çalıştır.

```
composer install
composer docker-compose:up
```

## Database Import
```
composer db:import
```

## Database Export
```
composer db:export
```

## PHPMyAdmin
```
composer phpmyadmin
```

## Silme
```
composer docker-compose:down
```
